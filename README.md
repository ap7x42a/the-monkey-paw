# The Monkey's Paw

Reusable Agent Skill for inherited systems and prior-agent artifacts.

Recover the intent behind what you inherited, distrust self-reports, classify the
artifact by how it serves or betrays that intent, and verify before keeping,
extending, deleting, or citing it.
